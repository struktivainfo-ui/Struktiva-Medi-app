struktiva v9.7 - Vercel Deploy

SO GEHT'S OHNE GITHUB:

1. Entpacke diese ZIP
2. Geh auf vercel.com → Login
3. "Add New..." → "Project"
4. Ziehe den GANZEN Ordner hier rein (Drag & Drop)
5. Deploy → Fertig!

Deine URL: https://struktiva-xxx.vercel.app

PUSH AKTIVIEREN:
- Auf Handy öffnen
- "Zum Home-Bildschirm" hinzufügen
- App öffnen → Einstellungen → "Push aktivieren"
- Erlauben → Fertig

Dateien:
- index.html = deine App
- manifest.json = PWA Daten
- sw.js = Push Service Worker
- vercel.json = Config
